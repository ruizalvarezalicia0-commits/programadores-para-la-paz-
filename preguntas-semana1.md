Pregunta 1 — Selección múltiple

¿Cuál de las siguientes opciones describe mejor qué es la ciudadanía digital?



A. El uso de internet únicamente para entretenimiento

**B. El uso responsable de la tecnología para informarse, participar y compartir información de manera ética**

C. El uso de redes sociales sin restricciones

D. El uso exclusivo de software comercial



**La respuesta correcta es la B. ya que se debe utilizar el internet de forma respetuosa y responsable.**



Pregunta 2 — Selección múltiple

¿Cuál es la razón principal por la que es importante verificar la información antes de compartirla en internet?



A. Para que la publicación tenga más reacciones

B. Para evitar consumir más datos móviles

**C. Para evitar difundir información falsa o engañosa**

D. Para que la publicación sea más extensa



**La respuesta correcta es la C. se debe verificar toda la información para no compartir información falsa o engañosa.**



Pregunta 3 — Selección múltiple

¿Cuál de las siguientes opciones define mejor un algoritmo?



A. Una aplicación móvil

**B. Una lista de pasos ordenados para resolver un problema**

C. Una carpeta del computador

D. Una red social



**La respuesta correcta es la B. es una serie de instrucciones organizadas y clara para resolver una tarea.**
